# Calculadora de IMC

Projeto em React  + Typescript para uma calculadora de Índice de Massa Corpórea(IMC).

Desenvolvido no módulo 4 de ReactJS do curso B7Web.

### Link para acesso:
https://react-calc-imc-nine.vercel.app/

### Instalação:
- `npm install`

### Parar inicializar:
- `npm start`

### Requisitos:
- NodeJS

- ReactJS
